# ✨ Shiny Pokédex Tracker Ultimate Edition

Uma Pokédex offline focada em colecionadores de Pokémon e caçadores de shinies.

O projeto nasceu como uma simples checklist para completar a Pokédex de Kanto e evoluiu para um tracker completo da National Dex, com suporte a regiões, Living Shiny Dex e integração com o Serebii.

## 🌟 Funcionalidades

* 📖 National Dex completa (até Pecharunt)
* ✨ Living Shiny Dex
* 🌎 Filtro por região

  * National
  * Kanto
  * Johto
  * Hoenn
  * Sinnoh
  * Unova
  * Kalos
  * Alola
  * Galar
  * Paldea
* 📦 Próximo Pokémon para capturar
* ✨ Próximo Shiny para obter
* 📋 Lista dos próximos 5 Shinies
* 🔍 Busca por nome
* 💾 Exportação de backup em JSON
* 📂 Importação de backup em JSON
* 📍 Atalho para a página do Pokémon no Serebii
* 🗃️ Organização por Boxes
* 💻 Funciona totalmente offline

## 🚀 Tecnologias

* HTML5
* CSS3
* JavaScript Vanilla
* LocalStorage

Nenhum framework ou biblioteca externa é necessário.

## 💾 Como funciona o salvamento

Todo o progresso é armazenado localmente utilizando o LocalStorage do navegador.

Isso significa que:

* Seu progresso permanece salvo entre sessões.
* Não é necessário criar conta.
* Não existe banco de dados.
* Os dados ficam armazenados apenas no navegador utilizado.

Recomendamos criar backups periódicos utilizando a opção Exportar JSON.

## 📂 Backup e Restauração

### Exportar

1. Clique no botão ⚙️
2. Clique em 💾 Exportar JSON
3. Salve o arquivo em um local seguro

### Importar

1. Clique no botão ⚙️
2. Selecione o arquivo JSON previamente exportado
3. O progresso será restaurado automaticamente
